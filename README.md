# Eudaimonia OS v3.0

Sistema personal de alto rendimiento — gamificación + filosofía estoica + psicología del comportamiento.

---

## Filosofía

> *"Forged in fire. Built in silence. Delivered in results."*

El sistema convierte disciplina diaria en identidad. No es una lista de tareas — es un motor de transformación personal sostenido durante **1 año** (objetivo: 5 500–6 000 XP acumulados).

---

## Sistema de XP

### Principio: XP = Esfuerzo real × Impacto

| Tier | XP | EC | Descripción |
|------|----|----|-------------|
| **Micro** | 1–2 | 0 | Hábitos de identidad base |
| **Progreso** | 3–5 | +1 | Crecimiento activo |
| **Alto Impacto** | 6–10 | +2 a +3 | Transformación real |

**Promedio diario esperado:** 10–15 XP  
**Fin de semana:** 35–40 XP

## Economía — Euda-Credits (EC)

- **Micro** → 0 EC
- **Progreso** → +1 EC
- **Alto Impacto** → +2 a +3 EC

Los EC se canjean en la **Tienda de Recompensas** (`/recompensas`).

### Recompensas de ejemplo

| Recompensa | EC | Nivel | Cooldown |
|-----------|-----|-------|----------|
| Ropa Nike | 50 | 3 | 30 días |
| Libro técnico | 30 | 2 | 7 días |
| Kindle | 120 | 4 | 90 días |
| Salida / Experiencia | 80 | 5 | 30 días |
| Apple Watch | 300 | 7 | 180 días |
| Viaje | 500 | 10 | 365 días |

Puedes agregar/editar recompensas directamente desde la tienda.

## Clasificación Diaria

Evaluación automática al final de cada día:

| Rango | Condición |
|-------|-----------|
| 🪨 **Carbón** | <7 XP o sin acciones de progreso |
| ⚔️ **Hierro** | 8–15 XP + ≥2 categorías |
| 🥇 **Oro** | 16+ XP + ≥3 categorías |
| 💎 **Diamante** | 20+ XP + ≥1 acción Alto Impacto |

---

## Sistema de Streaks

| Racha | Bonus |
|-------|-------|
| 7 días | +5% XP |
| 30 días | +10% XP |

Si se rompe: se pierde el multiplicador, **no** el XP acumulado.

---

## Bonus por Combo

- **LOGOI + HEGEMONIKON + PAIDEIA** en el mismo día → +3 XP automático
- **5 categorías** en un día → +5 XP automático
- **Sábado completo** (3 bloques) → +4 XP bonus
- **Domingo completo** (todas las fases) → +5 XP bonus
- **Sábado + Domingo** ambos completos → +8 XP global total

---

## Regla de Balance

Si una categoría está por debajo del 60% del promedio semanal:  
→ acciones de esa categoría reciben **+20% XP** hasta equilibrar.

No bloquea niveles — incentiva el equilibrio.

---

## Días Vacíos Controlados

Después de 3 días intensos → permitido 1 día ligero sin romper streak.  
Se activa desbloqueando el badge **"Maestro del Flow"**.

## Stack Técnico

- **Backend:** Python 3 + Flask
- **Base de datos:** SQLite local + Turso (híbrido async)
- **Frontend:** HTML + CSS custom + Lucide icons + Vanilla JS
- **Layout:** Sistema de diseño propio (dark theme, tokens CSS)
- **Deploy:** Railway (variable `DATABASE_PATH` para volumen persistente)

### Variables de entorno (opcional, para Turso)

```
TURSO_DATABASE_URL=libsql://tu-db.turso.io
TURSO_AUTH_TOKEN=tu-token
DATABASE_PATH=/data/pipeline.db
```
